### Logo Specifications

**Primary Logo Elements:**
- **Icon**: Brain/Neural Network symbol (represents memory and AI)
- **Text**: "Memory Connector" in modern, clean sans-serif font
- **Color**: Primary blue (#2563eb) with white or dark gray text

**Logo Variations Needed:**
1. **Full Logo**: Icon + Text (horizontal)
2. **Icon Only**: Brain symbol (for social media profiles)
3. **Text Only**: "Memory Connector" wordmark
4. **Dark Mode**: White/light version for dark backgrounds
5. **Monochrome**: Single color version for print

**Logo Usage:**
- Minimum size: 100px width for full logo
- Clear space: 2x icon size around logo
- Never distort or rotate logo
- Always use high-resolution versions (SVG preferred)

### Color Palette

**Primary Colors:**
- **Blue**: #2563eb (Primary brand color)
- **Dark Blue**: #1e40af (Hover states, emphasis)
- **Light Blue**: #3b82f6 (Accents, highlights)

**Supporting Colors:**
- **Green**: #10b981 (Success, positive actions)
- **Red**: #ef4444 (Errors, warnings)
- **Yellow**: #f59e0b (Alerts, important notices)
- **Gray Scale**: 
  - Dark: #111827 (Primary text)
  - Medium: #6b7280 (Secondary text)
  - Light: #f9fafb (Backgrounds)

**Usage Guidelines:**
- Use blue for primary CTAs and branding
- Use green sparingly for success states
- Maintain high contrast for accessibility
- Avoid using more than 3 colors in a single design

### Typography

**Primary Font**: Modern sans-serif (e.g., Inter, Poppins, or system font stack)
- **Headings**: Bold, 24-48px
- **Body**: Regular, 16-18px
- **Captions**: Regular, 12-14px

**Font Stack**: 
- Primary: System font stack for performance
- Fallback: Arial, Helvetica, sans-serif

### Visual Style

**Design Principles:**
- **Clean & Minimal**: Focus on content, minimal distractions
- **Modern**: Contemporary design with rounded corners
- **Friendly**: Approachable, not intimidating
- **Professional**: Trustworthy and reliable
- **Mobile-First**: Optimized for small screens

**Image Style:**
- Bright, clear screenshots of the app
- Real-world usage scenarios
- Diverse people using the app
- Clean backgrounds with subtle gradients
- Avoid cluttered or busy compositions

- **Resolution**: 1920x1080 (Full HD) minimum
- **Frame Rate**: 30fps
- **Format**: MP4 (H.264 codec)
- **Aspect Ratios**: 
  - 16:9 for YouTube, Facebook
  - 9:16 for TikTok, Instagram Stories
  - 1:1 for Instagram Feed
- **Audio**: 48kHz, stereo
- **File Size**: Under 100MB for web upload

**Content Guidelines:**
- **Hook**: Capture attention in first 3 seconds
- **Message**: Clear value proposition within 10 seconds
- **Visual**: Show the app in action, not just talking about it

- **Branding**: Logo visible for at least 2 seconds
